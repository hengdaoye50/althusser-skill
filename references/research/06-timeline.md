# 阿尔都塞完整时间线

> 调研日期：2026-04-26

---

## 一、主要时间线

| 时间 | 事件 | 对思想的影响 |
|------|------|------------|
| 1918 | 出生于阿尔及尔比曼（法属阿尔及利亚） | 殖民地经验、边缘性——在法语哲学中心始终是"外来者" |
| 1930s | 里昂名校教育、参与天主教青年运动 | 早期宗教经验后来被理论化为"意识形态传唤"的原型 |
| 1939 | 入读巴黎高师（ENS），准备哲学教师资格考试 | 进入法国学术核心 |
| 1940 | 被征入伍，德军入侵后被俘 | **关键转折**：战俘营5年——深度阅读、与工人农民同处，暴露了理论—实践的裂缝 |
| 1940-1945 | 德国战俘营（Schleswig, Stalag X-A） | 对"主体性"和"共同体"的原初经验，但从未被理论化 |
| 1945-1948 | 回到高师完成学业，通过教师资格考试 |  |
| 1948 | 加入法共（PCF）；被任命为高师哲学辅导教师（caïman） | 开始在党内外两条线上的双重身份：理论异端 + 组织忠诚 |
| 1948-1960 | 早期写作：关于孟德斯鸠、关于青年马克思的论文；与福柯的交往 | 福柯是同一高师的学弟/同事；杜梅泽尔是阿尔都塞推荐给福柯的 |
| 1956 | 赫鲁晓夫秘密报告；匈牙利事件 | **关键转折**：斯大林教条主义破产，马克思主义面临理论真空——阿尔都塞的理论介入由此开始 |
| 1960s初 | 接触巴什拉、康吉莱姆、拉康 | 认识论断裂（巴什拉）、科学史认识论（康吉莱姆）、主体问题（拉康）——三大智识来源汇聚 |
| 1961 | "矛盾与多元决定"发表 | 第一篇核心方法论论文，暗中回应萨特 |
| 1962 | "关于唯物辩证法"发表 | 理论实践概念的系统阐述 |
| 1963 | "马克思主义不是人道主义"发表 | 反人道主义纲领 |
| 1964-1965 | 组织《读〈资本论〉》研讨班 | **核心事件**：征候阅读法的集体实践，影响一代法国哲学家 |
| 1965 | 《保卫马克思》《读〈资本论〉》出版 | **理论爆发**：一跃成为法国马克思主义最具影响力的理论家 |
| 1967 | "哲学与科学家的自发哲学"讲座 | 开始修正立场：从"理论实践"转向"哲学是阶级斗争" |
| 1968 | 五月风暴；住院缺席 | **个人与理论的双重危机**：精神崩溃缺席革命；理论框架面对主体性爆发的无力 |
| 1968 | 意大利电台访谈 | 提出"哲学是理论中的阶级斗争"——正式转向 |
| 1969 | 《列宁与哲学》出版 | 转向后的核心文本 |
| 1970 | 《意识形态与意识形态国家机器》发表 | **影响最深远的单篇文本**，但计划续篇从未完成 |
| 1970s | ISA理论在英语世界传播 | 文化研究（霍尔）、电影理论（博德里）、批判理论全面接纳ISA框架 |
| 1973 | 《答约翰·刘易斯》出版 | 少数正面回应批评的案例，方式是"诊断"而非"讨论" |
| 1974 | 《自我批评材料》 | 承认"理论主义"但拒绝放弃反人道主义 |
| 1974 | 朗西埃出版 La Leçon d'Althusser | 最亲密学生的决裂——对阿尔都塞理论主义的系统性批判 |
| 1978 | E.P. 汤普森《理论的贫困》 | 英语世界最激烈的攻击；阿尔都塞完全未回应 |
| 1980.11.16 | 勒死妻子埃莱娜 | **最深的断裂**：因精神疾病"不胜诉"；对思想界的影响远超学术争论 |
| 1980-1987 | 精神病院中的写作 | 发展"偶然唯物主义"——对自身早期理论的根本性修正 |
| 1985 | 写完自传《来日方长》 | 他在自己否定主体的理论中，写出了最具主体性的文本 |
| 1987 | 离开精神病院 | |
| 1990.10.22 | 去世于巴黎 | |
| 1992 | 《来日方长》出版 | 引发新一轮"理论与生命"争论 |
| 1994 | Écrits philosophiques et politiques 出版 | 偶然唯物主义文本首次大规模面世 |
| 1995 | Sur la philosophie 出版 | |
| 2006 | Valence de la rupture 出版 | |

## 二、思想转折点因果链

```
1956 赫鲁晓夫秘密报告
  → 马克思主义理论真空
    → 阿尔都塞的"第三条路"（反教条主义 + 反人道主义）
      → 《保卫马克思》《读〈资本论〉》

1964-65 征候阅读的集体实践
  → 《读〈资本论〉》研讨班
    → 学生们的理论化和后续分歧

1967-68 对"理论实践"的自我质疑
  + 五月风暴催化
    → "哲学是理论中的阶级斗争"转向

1970 ISA与传唤理论
  → 留下"传唤失败"的空位
    → 齐泽克的拉康化/巴迪欧的事件本体论/朗西埃的感性分配

1974 自我批评 + 学生决裂
  → 承认"理论主义"但坚持核心立场

1980 悲剧事件
  → 精神病院写作
    → 偶然唯物主义——对"结构决定"的根本性修正
```

## 三、斯宾诺莎联系的三个阶段

| 阶段 | 时间 | 特征 |
|------|------|------|
| 隐性期 | 1960s | 斯宾诺莎的内在性因果隐含在"结构性因果"中，但未明言 |
| 显性期 | 1970s | 在未发表手稿中明确引用斯宾诺莎，尤其是"内在性"概念 |
| 激进化期 | 1980s | 偶然唯物主义中，斯宾诺莎的"实体"与马基雅维利的"偶然"结合 |
