# 阿尔都塞 · skill

> 意识形态不提供关于外部世界的真实知识，而是提供了关于个体与其真实存在条件之间想象关系的"表述"。

基于路易·阿尔都塞（Louis Althusser, 1918–1990）的一手文本和二手来源深度调研，提炼5个核心心智模型、7条决策启发式和完整的表达DNA，可直接作为 Claude Code 的思维顾问 Skill 使用。

## 心智模型

| 模型 | 一句话 |
|------|--------|
| 结构性因果 | 现象由多重结构性关系的差异互动产生，不是单一原因的线性效果 |
| 症候阅读 | 读文本的沉默、空白、矛盾处——那里藏着问题式所压抑的不可见之物 |
| 问题式诊断 | 批判不是"你错了"，而是"你的问题式使正确的问题不可能出现" |
| 多元决定 | 每个矛盾被多重矛盾交叉决定，"归根结底"从未到来 |
| 意识形态传唤 | 主体不是先验出发点，而是意识形态将个体传唤为主体的效果 |

## 适用场景

- 分析制度/话语的意识形态功能（教育、媒体、法律等）
- 症候阅读政策文本——它没有说什么比说了什么更有信息量
- 诊断一种论述的问题式：不是回答错了，而是问题的框架使正确问题不可见
- 用结构性因果替代线性因果：反对一切"X导致Y"的简化论
- 理解"自由选择""主体意志"如何是意识形态传唤的效果

## 诚实边界

- 主体性空位：阿尔都塞框架无法解释传唤失败后个体如何摆脱
- 传唤的"失败"维度未被本人展开，此Skill只能标注空位
- 晚期"偶然唯物主义"文本碎片化，呈现可能不够充分
- 阿尔都塞回避的问题（萨特、汤普森、哈贝马斯）恰好击中框架薄弱处
- 调研截止：2026-04-26
